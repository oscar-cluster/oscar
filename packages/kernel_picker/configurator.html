A Form

The configuration of a package is done by presenting the user with a very basic HTML form. The maintainer of the package needs to generate this HTML file. Below you will find some of the things you can do with a FORM. Note that there will eventually be extra stuff like lists of machines which will be generated at runtime and can be inserted by using a comment.
