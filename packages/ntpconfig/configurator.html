Select NTP Servers

The ntpconfig package sets up ntp (the Network Time Protocol) on your cluster so that all of your machines will be synchronized to the same time. However, we also would like the OSCAR server node's time to be synchronized with the rest of the world. In order to do that, we need to specify at least one public NTP server to which we can connect.

Below are three public NTP servers that should work just fine for most people. However, you can select/deselect these servers as you like. You can also add other servers to the list in the text box below. In the text box, enter one server per line.

NOTE:You must have at least one NTP server selected/entered or by default you get the three general use public NTP servers below.

General Use Public NTP Servers

ntp-1.cso.uiuc.edu
ntp0.cornell.edu
ntp.cmr.gov

Additional/Alternate NTP Servers

Enter additional FQDNs (fully qualified domain names) or IP addresses of NTP servers that you would like to use. Enter one server per line.
